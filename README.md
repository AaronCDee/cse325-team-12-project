# TaskTracker

TaskTracker is a Blazor Web App for students and professionals who need a simple way to create, update, complete, and filter personal tasks.

## Team Members

- Aaron Delate
- Alexandre Jiresse
- Boluwatife Omotoyinbo

## Current Stack

- .NET 10
- Blazor Web App
- SQLite for local development
- Entity Framework Core

## Repo Structure

- `TaskTracker.csproj`: main Blazor application project
- `Components`: Blazor UI components and pages
- `Data`: database context and data access setup

## Git Workflow

- `main` stays stable
- Create branches from `main` using `wk<week>-<name>`
- Current working branch for this task: `wk4-boluwatife`
- Merge changes through pull requests

See `CONTRIBUTING.md` for the team workflow.

## Getting Started

1. Install the .NET 10 SDK.
2. Restore dependencies:

```bash
dotnet restore
```

3. Run the app:

```bash
dotnet run
```
